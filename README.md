Expense Tracker
Автор: Мешалкин Платон Дмитриевич

Описание программы
Простое приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять расходы, фильтровать их по категории и дате, считать сумму за период и сохранять/загружать данные в формате JSON.

Как использовать
Запустите app.py.
Введите сумму, категорию и дату.
Нажмите «Добавить расход».
Для фильтрации — укажите параметры и нажмите «Фильтровать».
Для подсчёта суммы — укажите период и нажмите «Посчитать сумму».
Сохраняйте и загружайте данные через соответствующие кнопки.
Примеры использования
Добавить расход: сумма — 500, категория — «еда», дата — 2026-04-23.
Посчитать сумму за неделю.
Отфильтровать все расходы на «транспорт».
Сохранить данные и загрузить их при следующем запуске.
